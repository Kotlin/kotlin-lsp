# Neovim setup

1. [Install `kotlin-lsp` CLI](../README.md#install-kotlin-lsp-cli)
2. Make sure that the `kotlin-lsp` binary is on your `$PATH`
3. Install the official Neovim [`nvim-lspconfig` plugin](https://github.com/neovim/nvim-lspconfig)
4. Enable the LSP `vim.lsp.enable('kotlin_lsp')` in your `init.lua`
